# Vue Portifolio - Personal Website Template

This a simple ```VueJS``` template for make a developer's portifolio.

```
### Project setup
npm install

### Compiles and hot-reloads for development
npm run serve

### Compiles and minifies for production
npm run build


### Run your tests
npm run test

### Lints and fixes files
npm run lint

### Run your unit tests
npm run test:unit
```
